## 📁 Clone

With project clone, go file folder

```
$ cd rec
```

Install dependencies
[NodeJS](https://nodejs.org/en/)

Package manager yarn (not required)
[Download Yarn](https://yarnpkg.com/getting-started/install)

```
$ npm i or $ yarn
```

Run project

```
$ npm run start or $ yarn start
```
